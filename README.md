# Website Control Panel

This control panel allows you to manage your connected sites, featuring:

- **Push & Pull Mechanism**: Send or receive data from your connected sites.
- **Pop-up Ad Creation**: Dynamically create pop-up ads.
- **Lock/Unlock Controls**: Lock or unlock the entire site or specific pages.
- **Theme & Language Switching**: Change themes and languages across your sites.
- **API Key Generation**: Generate an API key for your control panel.
